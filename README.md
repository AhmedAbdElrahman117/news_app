## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AhmedAbdElrahman117/news_app.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd news_app
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```
